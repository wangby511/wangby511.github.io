# 前后自增运算符

前自增运算符改变了对象的状态并返回对象改变后的状态，不需要创建临时对象。

#### 前自增运算符的例子
```
MyOwnClass& operator++(){
    ++meOwnField;
    return (*this);
}
```

#### 后自增运算符重载的例子
```
MyOwnClass operator++(int)
{
    MyOWnCLass tmp = *this;
    ++(*this);
    return tmp;
}
//摘自https://zhuanlan.zhihu.com/p/54647542
```
